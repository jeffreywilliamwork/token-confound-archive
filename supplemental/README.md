# Supplemental Material

This directory holds adjacent material that is not necessary to understand the token-confound story.

Included here:
- `data/ds31-v22-32q-1_prefill.*`
- `data/ds31-v22-32q-1_multiseed_seed_*.json`
- `docs/METHODOLOGY_V2.md`
- `recovery/ds31_v22_partial_raw_recovery.*`

Why these are supplemental:
- the v2.2 forced-choice runs are about a different question
- the multiseed files do not bear on the hierarchy confound
- `METHODOLOGY_V2.md` is a post-confound protocol note, not part of the failure analysis itself
- the v2.2 raw recovery is a useful archival salvage pass, but only for a partial prefill subset

They are preserved for completeness, but an outside reader can ignore this directory and still understand what happened.
